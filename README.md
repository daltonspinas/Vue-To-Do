# Vue-To-Do
A to-do app with a Vue frontend

## Build Setup

``` bash
## cd into the server folder

# create a local database called "vue-todo"

# install dependencies
npm install

# run the seed file
npm run seed

#start the server
npm run start

## now cd into the client folder

# install dependencies
npm install

# fire up the app
npm run dev
will be hosted on localhost:8080

```
